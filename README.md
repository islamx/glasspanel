# GlassPanel 🌟

A modern, full-stack glassmorphic web application built with Next.js and PocketBase, featuring a beautiful glass panel design with Arabic and English language support. This project demonstrates a complete solution from UX design to backend integration, showcasing modern web development practices.

## 🚀 Demo

**Live Demo:** [https://glasspanel.vercel.app](https://glasspanel.vercel.app)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/islamz/glasspanel.git
cd glasspanel
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
```

4. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)
