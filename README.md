## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/islamz/glasspanel.git
cd glasspanel
```

2. **Install dependencies**
